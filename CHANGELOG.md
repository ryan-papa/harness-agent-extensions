# Changelog

## [0.18.0] - 2026-07-24

### Changed
- 독립 리뷰 **호출 1건당 타임아웃 10분(600초)** — rp-deck·rp-post 공통. 리뷰가 무한정 늘어져 파이프라인을 잡아먹는 것을 차단
  - Codex: Bash 툴 `timeout: 600000`(ms)로 하드 강제 (⛔ 값 상향 금지)
  - Claude 서브에이전트: Agent 툴에 타임아웃 인자가 없으므로 프롬프트에 "10분 내 반환" 상한 명시(미완이면 그 시점까지의 평가만 반환) + 메인이 초과 판단 시 결과 없이 진행
  - 적용 단위는 **건당** — 합산 아님. rp-deck은 R1/R2 × Claude/Codex 각각 10분
  - 초과 시 **비차단 즉시 스킵**(⛔ 재시도·연장 없음), 1줄 로그(`타임아웃[Codex-R2]: 600s 초과, 스킵`) 후 남은 리뷰어 결과로 진행. 양쪽 다 타임아웃이면 그 라운드는 지적 0건 처리(rp-post는 리뷰 요약에 명시)

## [0.16.0] - 2026-07-17

### Changed
- rp-deck 게시판 index — 좌우 2-pane 독립 스크롤 + 모바일 세로 레이아웃 (`update_index.py`)
  - 데스크톱: 100dvh 고정, 좌측 트리 카드·우측 리스트 카드가 각각 내부 스크롤, 리스트 헤더 sticky, 페이지 자체 스크롤 없음
  - 모바일(≤860px): 2-pane 해제·페이지 전체 스크롤, 행을 제목·설명·날짜 세로 스택(날짜 좌측 정렬·설명 2줄), 컬럼 헤더 숨김, 좌우 패딩 대칭(16px)
  - 트리·리스트 pane 스크롤바 auto-hide — idle 시 투명, scroll 이벤트 700ms 동안·호버 시 노출 (`scrollbar-width/-color` + `::-webkit-scrollbar` 병행)

## [0.15.0] - 2026-07-16

### Changed

  - 좌측: 독립 "전체" 버튼 + 뎁스별 중첩 디렉토리 트리(개수 뱃지·접기/펼치기)
